应用场景图片准备说明
====================

请准备以下五张应用场景图片，并放置在当前目录中：

1. creative-display.jpg - 创意显示场景
   参考：https://www.kinglight.com/creative-display.html
   建议：透明屏或异型屏的实际应用图片，如建筑立面、艺术装置等

2. commercial-display.jpg - 商业显示场景
   参考：https://www.kinglight.com/commercial-display.html
   建议：商场、机场、会议室等场所的大屏幕显示应用图片

3. stage-performance.jpg - 舞台演绎场景
   参考：https://www.kinglight.com/stage-performance.html
   建议：演唱会、舞台剧、活动庆典等使用LED显示屏的场景图片

4. home-display.jpg - 家用显示场景
   参考：https://www.kinglight.com/home-display.html
   建议：家庭影院、客厅等使用LED显示屏的场景图片

5. movie-production.jpg - 电影制播场景
   参考：https://www.kinglight.com/movie-production-and-broadcasting.html
   建议：电影制作、虚拟拍摄、广播电视等专业LED应用场景图片

图片要求：
- 分辨率建议至少1920x1080或更高
- 图片应清晰、专业，展示LED显示屏在各场景中的实际应用
- 图片应有版权或使用授权

临时解决方案：
如果暂时无法获取这些图片，可以使用一些通用的LED显示应用场景图片作为替代，
但最终应当使用与众普光产品相关的实际应用场景图片。
